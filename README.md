<h2>Features</h2>
<ul>
  <li>Makes small mouse movements to simulate real mouse clicks</li>
  <li>Allows you to create a pattern of clicks to avoid detection algorithms</li>
  <li>Option to set custom hotkeys</li>
</ul>
